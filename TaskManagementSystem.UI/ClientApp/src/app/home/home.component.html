Overview:

The current system for managing basic tasks. System will allow to users, adding, listing and

updating tasks and also able to adding comments to it. Users will be able to assign task to other users and set action date for  task via comments. 

 

Goals:

The system will include web based application with:

  • Multi layered architecture (Clean Architecture)
  • Client App with Angular + TypeScript + Html
  • Postgresql 
  • .Net Core & C#

 

Requirements:

  • Task Dashboard
  • Task forms for creation and editing
  • Comments form  creation and deleting
  • Back-end services for CRUD operations
